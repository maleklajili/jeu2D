# jeu2D

Ce projet est un jeu vidéo 2D développé avec Unity.

## Structure du projet
- **Animation/** : Contrôleurs et animations pour les personnages, ennemis et objets.
- **Prefabs/** : Préfabriqués pour les éléments du jeu.
- **Scenes/** : Scènes du jeu.
- **script/** : Scripts C# pour la logique du jeu.
- **Assets graphiques** : Dossiers contenant les sprites, tilesets, UI, etc.

## Démarrage rapide
1. Ouvrez le dossier dans Unity.
2. Lancez la scène principale depuis le dossier `Scenes`.
3. Jouez ou modifiez selon vos besoins.
